# VidScribe - AI-Powered YouTube Video Summarizer

![VidScribe Logo](https://img.shields.io/badge/VidScribe-YouTube%20AI%20Summarizer-blue?style=for-the-badge&logo=youtube)

## ğŸš€ Getting Started

### Prerequisites

- **Python 3.9+**
- **Node.js 18+**
- **Redis Server** (Local or Managed like Upstash)
- **Google AI Studio Key** (for Gemini)
- **Google Cloud Console Key** (for YouTube Data API v3)

### 1. Backend Setup

```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Fill in your API keys in .env
python -m app.main
```

### 2. Frontend Setup

```bash
cd frontend
npm install
cp .env.example .env.local
# Ensure NEXT_PUBLIC_API_URL points to your backend
npm run dev
```

The application will be live at `http://localhost:3000`.
